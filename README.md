# inkmango start

## tech

- graphql
- prisma
- express
- react

## quick start

- Install frontend
  `npm install`
- Install server

`cd server`
`npm install`

- Start server
  `npm run dev`
- Start react
  `cd .. npm react-start`

### Ports

GraphQL - 4000
Prisma - 4466
Express - 3000
React - 3001
